# Assignment 1

The purpose of this assignment is to get comfortable with the following concepts:

- Variable assignment and usage
- Control Flow Constructs:
  - if
  - for loop
- Array manipulation
- Functions:
  - Defining named functions
  - Adding parameters to functions
  - Calling functions
- Core data types:
  - Number
  - String
  - Object
  - Array
  - Function
  - Boolean
